    CSV formats

    You can upload the CSV file generated by Edulink One

    To generate the file in Edulink One, when logged in:
    1. Select 'Analytics'.
    2. Then select the following options:
      • Report: Achievement overview
      • From: 04/09/2023 To: 'current date'
      • Group By: Student
      • Y axis: Points
    3. No optional filters.
    4. Then click on 'UPDATE'
    5. Then 'EXPORT DATA', save the file and upload that file here.

    Or you can upload either of the following files:

    • Raw export: columns Name, Points, optional Form Group, Year Group.
    • Converted: columns first_name, last_name, merits, optional form_name, year_group.

    Multiple rows per pupil are aggregated. APs never decrease.

    Examples:

    Name,Points,Form Group,Year Group
Alice Smith,5,7OHA,7
Bob Jones,3,7MDU,7
    first_name,last_name,merits,form_name,year_group
Alice,Smith,5,7OHA,7
Bob,Jones,3,7MDU,7
